# Landing Page

This project is part of the curriculum from The Odin Project.
The goal of this project is to build a **static landing page** using HTML and CSS while applying fundamental layout and styling concepts.

## Project Overview

The landing page is designed to replicate a provided design layout. It focuses on structuring content properly with HTML and styling it using CSS, particularly using **Flexbox** to create responsive layouts.

This project helped reinforce my understanding of:

* HTML page structure
* CSS styling
* Flexbox layout
* Organizing webpage sections

## Features

* Header with navigation links
* Hero section with call-to-action
* Information cards section
* Quote/Testimonial section
* Call-to-action banner
* Footer

## Built With

* HTML5
* CSS3
* Flexbox
